# Used Car Price Prediction

A Machine Learning web application built with Streamlit that predicts the price of used cars based on various features.

## Features
- Brand and Model Selection
- Car Age Calculation
- KM Driven Input
- Fuel Type Selection
- Transmission Type
- Owner Type
- Feature Importance Visualization
- Analytics Dashboard

## Tech Stack
- Python
- Streamlit
- Scikit-Learn
- Pandas
- NumPy
- Matplotlib

## Run Locally

```bash
pip install -r requirements.txt
streamlit run predict.py
```

## Project Structure

- predict.py
- train.py
- preprocess.py
- feature_importance.py
- requirements.txt
- car_model.pkl
- used_car_dataset.csv
